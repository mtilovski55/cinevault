🎬 CineVault

A modern movie catalog web app built with React + Vite.

🚀 Features
	•	🔐 User authentication (register & login)
	•	🎥 Browse movie catalog
	•	🔍 Search movies by title
	•	🎭 Filter by genre
	•	↕️ Sort by title, year, and rating
	•	⭐ Add movies to watchlist
	•	✅ Mark movies as watched
	•	➕ Create new movies
	•	💬 Clean UI with validation and feedback states

🛠 Tech Stack
	•	React (Hooks)
	•	React Router
	•	Vite
	•	JSON Server (mock backend)
	•	CSS (custom styling)

📦 Installation
  npm install
  npm run dev
