🎬 CineVault

CineVault is a full-stack movie catalog web app that allows users to browse, manage, and track their personal movie collection.

Built with React and a REST API (json-server), it demonstrates authentication, state management, and real-world UI/UX patterns.

✨ Key Features
- User authentication (register & login)
- Browse and search movies
- Filter and sort catalog
- Add/remove movies from watchlist
- Mark movies as watched
- Create new movie entries
- Persistent state via API

🛠 Tech Stack
- React (Hooks)
- React Router
- Vite
- JSON Server (mock backend)
- CSS (custom styling)

📦 Installation
  npm install
  npm run dev
